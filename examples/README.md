# Exmaples

Here you can find examples on how to use this library with different simulators and test your models. Please refer to the README.md in all example folders for further insturctions and documentations on the examples.

## Installation

Make sure you installed all dependencies using `pip install -r requirements.txt` for each example.
Make sure to read the `README.md` and docs on `perturbationdrive`.

## Self Driving Sandbox Donkey

Example on running the PerturbationDrive library using the [Self Driving Sandbox Donkey Simulator](https://docs.donkeycar.com/guide/deep_learning/simulator/). Please note, that the concrete simualtor instance used here is this fork of the project [fork of the project](https://github.com/HannesLeonhard/self_driving_sandbox_donkey/tree/crossroad/sdsim).

## Udacity Simulator

Example on running the PerturbationDrive library using the [udacity](https://github.com/udacity/self-driving-car-sim) simulator.

## OpenSBT

Examples on running the PerturbationDrive library and finding scenarios using [Open SBT](https://git.fortiss.org/opensbt).

## Models

Example implenetation of the `ADS` interface to test the simulator examples.
